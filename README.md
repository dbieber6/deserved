# Deserved — website

Static site for the **Deserved** iPhone app, served via GitHub Pages.

- Landing page: <https://dbieber6.github.io/deserved/>
- Privacy policy: <https://dbieber6.github.io/deserved/privacy/>

Plain static HTML — no build step. Edit the `.html` files and push to `main`;
GitHub Pages redeploys automatically. `.nojekyll` disables Jekyll processing so
the files are served exactly as written.
